# eShell 1.1.0

Release date: 2026-04-03

## Highlights

- Added server status components and wallpaper management to make the workbench more customizable and informative.
- Improved Ops Agent reliability with better error handling, clearer chat feedback, a unified shell tool flow, and a more stable react loop.
- Upgraded SFTP workflows with transfer progress events, cancel support, download-to-local support, and a compact transfer queue overlay.
- Stabilized SSH PTY input handling and restored custom window titlebar drag behavior on Windows.
- Continued backend refactoring for commands, storage, and server operations, plus refreshed development documentation.
