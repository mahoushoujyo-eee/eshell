# eShell 1.3.0

Release date: 2026-04-14

## Highlights

- Moved approval mode out of AI profile configuration into a global conversation-level setting.
- Rebuilt the approval mode control as a compact composer action with localized `Approval` and `Full Access` labels.
- Added compatibility migration for older AI profile data so legacy per-profile approval mode is preserved as the new global setting.
- Updated the custom desktop title bar to use macOS-style traffic lights on macOS while keeping the existing Windows/Linux control layout.
- Added title bar platform debug helpers so the macOS layout can be previewed and verified without changing host code.

The release was validated with:

## Upgrade Notes

- Existing `.eshell-data/ai_profiles.json` files with legacy profile-scoped approval settings are migrated to the new app-level approval mode automatically.
- Runtime data remains under `.eshell-data/` in the project root.
- `docs/refer_proj/` remains local reference material and is not part of the release contents.
